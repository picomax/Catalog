# 초간단 Catalog 앱

강의/학습 용으로 제작한 초간단 카달로그 앱

## 주요 기능
CatalogInfo.plist 파일에 여행지와 이미지 리소스 파일에 대한 정보가 있고,
이 정보들을 로드하여 리스트와 상세페이지에 출력합니다.
> CatalogInfo.plist
